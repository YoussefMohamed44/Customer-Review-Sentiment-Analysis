# Customer Review Sentiment Analyzer

A simple Python-based CLI tool that predicts the sentiment of customer reviews using a trained machine learning model.

## Features
- Predicts sentiment as **positive**, **neutral**, or **negative**.
- Displays prediction **confidence percentage**.
- Cleans input reviews by removing stopwords and non-alphabetic characters.
- Runs entirely in the command line.

## Requirements

- Python 3.8 or higher
- Required libraries (see `requirements.txt`):
  - `nltk`
  - `scikit-learn`
  - `numpy`
